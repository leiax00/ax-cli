# WezTerm 快捷键速查

Leader 键：`Ctrl+A`（5 秒超时，与 tmux 兼容）

## 标签页

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` → `a` | 发送原生 `Ctrl+A`（tmux 兼容） |
| `Ctrl+A` → `c` | 新建标签页 |
| `Ctrl+A` → `n` | 下一个标签页 |
| `Ctrl+A` → `p` | 上一个标签页 |
| `Ctrl+A` → `w` | 关闭当前标签页（需确认） |
| `Ctrl+A` → `1~5` | 切换到第 1~5 个标签页 |

## 分屏

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` → `\` | 水平分屏（左右） |
| `Ctrl+A` → `-` | 垂直分屏（上下） |
| `Ctrl+A` → `h/j/k/l` | 切换到 左/下/上/右 面板 |
| `Ctrl+A` → `H/J/K/L` | 调整面板大小（左/下/上/右，步长 5） |
| `Ctrl+A` → `z` | 面板全屏/还原切换 |
| `Ctrl+A` → `x` | 关闭当前面板（需确认） |

## 复制粘贴

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` → `[` | 进入复制模式 |
| `Ctrl+Shift+V` | 从剪贴板粘贴 |
| 鼠标左键选中 | 自动复制到剪贴板 |
| 鼠标右键 | 从剪贴板粘贴 |

## 滚动

| 快捷键 | 功能 |
|--------|------|
| `Shift+PageUp` | 向上翻一页 |
| `Shift+PageDown` | 向下翻一页 |
| `Ctrl+A` → `u` | 向上翻半页 |
| `Ctrl+A` → `d` | 向下翻半页 |

## 字体大小

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+=` | 增大字体 |
| `Ctrl+-` | 缩小字体 |
| `Ctrl+0` | 重置字体大小 |

## 窗口

| 快捷键 | 功能 |
|--------|------|
| `F11` | 切换最大化（自定义 action_callback） |

## 其他

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` → `r` | 重载配置文件 |
